# bookstore_client

> Bookstore client app written in Kotlin.

## Requirements
* Android minSdkVersion: 21
* Kotlin version: 1.3.61

## Technologies and libraries
* **Java 1.8**
* **Android Studio**
* **Kotlin**

## Features
* Drawer based menu
* Bottom navigation bar for administrator's panel
* Dynamic lists made with RecyclerViews
